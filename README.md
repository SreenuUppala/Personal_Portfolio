# Personal Portfolio Website

## Overview

This is my personal portfolio website built to showcase my skills, projects, achievements, and contact information.

The portfolio serves as a central place for recruiters, developers, and collaborators to learn more about my work and technical background.

## Features

* Responsive design
* About Me section
* Technical Skills section
* Projects showcase
* Resume download option
* Contact information
* GitHub and LinkedIn integration

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub
